class HTMLHelper¶

Utility class for all HTML drawing classes

property formatOptions¶
Type

array

Option values related to the generation of HTML output. Recognized options are: fmtDepth, integer. The current indent depth. fmtEol, string. The end of line string, default is linefeed. fmtIndent, string. The string to use for indentation, default is tab.
