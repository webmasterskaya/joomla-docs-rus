Joomla\CMS\Form\FormHelper¶

class FormHelper¶

Form’s helper class. Provides a storage for filesystem’s paths where Form’s entities reside and methods for creating those entities. Also stores objects with entities“ prototypes for further reusing.

property paths¶
Type

array

Scope

protected

Array with paths where entities(field, rule, form) can be found.

Array’s structure:

paths: {ENTITY_NAME}: - /path/1 - /path/2

property prefixes¶
Type

string

Scope

protected

The class namespaces.

property entities¶
Type

array

Scope

protected

Static array of Form’s entity objects for re-use. Prototypes for all fields and rules are here.

Array’s structure: entities: {ENTITY_NAME}: {KEY}: {OBJECT}
